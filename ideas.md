<text>
**Design Movement**: 따뜻한 에디토리얼 미니멀리즘 (Warm Editorial Minimalism)

**Core Principles**:
1. 사람의 온기가 느껴지는 크림/테라코타 팔레트 — 차갑고 딱딱한 기업 이미지 완전 배제
2. 명조체(Serif)와 산세리프의 대비 — 전문성과 친근함의 균형
3. 비대칭 레이아웃 — 좌측 텍스트 + 우측 이미지의 역동적 구성
4. 넓은 여백과 숨쉬는 공간 — 프리미엄하고 여유로운 인상

**Color Philosophy**:
- Background: `#F9F6F0` (크림/아이보리) — 완전한 흰색보다 눈이 편안하고 따뜻함
- Primary CTA: `#E87A5D` (소프트 코랄/테라코타) — 생동감과 열정, 행동 유도
- Secondary: `#8A9A86` (워머 세이지 그린) — 안정감, 성장, 신뢰
- Text: `#2C2416` (딥 브라운) — 순수 블랙보다 따뜻하고 부드러운 대비
- Accent: `#D4B896` (웜 샌드) — 섹션 구분 및 배경 변주

**Layout Paradigm**:
- 비대칭 히어로: 좌측 2/3 텍스트 + 우측 1/3 이미지 (또는 반대)
- 지그재그(Zigzag) 섹션 배치로 단조로움 탈피
- 숫자 카운터 섹션은 풀 와이드 배경색 변주
- 카드형 서비스 섹션은 3열 그리드

**Signature Elements**:
1. 손글씨 느낌의 이탤릭 강조 텍스트 (Playfair Display Italic)
2. 얇은 선(divider)과 원형 장식 요소
3. 섹션 전환 시 부드러운 페이드인 애니메이션

**Interaction Philosophy**:
- 스크롤 시 요소들이 아래에서 위로 부드럽게 등장 (Fade Up)
- 버튼 호버 시 배경색 전환 + 미세한 스케일 업
- 카드 호버 시 그림자 강화 + 미세한 상승 효과

**Animation**:
- framer-motion 활용한 스크롤 트리거 애니메이션
- 숫자 카운터 애니메이션 (0에서 목표값까지 카운트업)
- 히어로 텍스트 순차적 등장 (stagger effect)

**Typography System**:
- Display/Hero: Noto Serif KR (명조, 굵게) — 권위와 신뢰
- Heading: Pretendard (산세리프, 600-700) — 명확성
- Body: Pretendard (산세리프, 400) — 가독성
- Accent/Quote: Noto Serif KR Italic — 감성적 강조
